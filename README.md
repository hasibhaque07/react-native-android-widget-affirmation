# 🚀 Getting Started

First, install all dependencies:

```bash
   npm install
```

# 📱 Running the Android Widget

This project uses react-native-android-widget, which requires native setup.  
So you need to prebuild the project before running it.

## 1. Prebuild the project

```bash
   npx expo prebuild -p android
```

This will generate the native Android project.

## 2. Run the app on Android

```bash
   npx expo run:android
```
